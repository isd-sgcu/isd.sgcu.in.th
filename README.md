# isd.sgcu.in.th

This repository is source code for isd.sgcu.in.th, the website for ISD SGCU.

## Tech Stack

This website currently on have frontend since the content does not change very often.

- Astro
- Svelte
- TailwindCSS

## Prerequisites

We recommend using [asdf](https://asdf-vm.com/) to manage your development environment. The required versions are:

- Node.js 24.x LTS
- pnpm 11.x

You can install the required versions using asdf with the following commands:

```bash
asdf plugin add nodejs
asdf plugin add pnpm
asdf install
```

## Before creating pull request

- Make sure your code is formatted and have no lint issues, run `pnpm format` to format code.
